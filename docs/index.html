Introdução¶

Hoje em dia fcil conseguir dados. Porm, conseguir dados rotulados nem tanto. O processo de rotulao pode ser caro e demorado. Nesta situao, como podemos otimizar esse trabalho e escolher os dados certos para rotular? Como garantir que a amostra de dados a serem rotulados no seja redundante ou represente bem a distribuio dos dados como um todo?

Para isso, existe uma tcnica chamada Active Learning. O diagrama de como feito essa tcnica mostrado a seguir. O princpio escolher de forma inteligente a melhor amostra a ser rotulada com base no seu modelo e nas suas incertezas.

Como escolher os melhores dados a serem rotulados? Ao pegar uma amostra em uma regio incerta (pode ser visto em um espao vetorial mais pra frente), certamente vamos obter mais informaes do que pegar uma amostra aleatria. Isso chamado de pool-based sampling. A ideia pegar os dados que esto em uma regio incerta e fazermos a rotulao neles. Mas como pegar esses dados na regio incerta? Para isso vamos adotar a estratgia de Uncertainty Sampling.

Essa estratgia mensurada pela frmula:

U ( x ) = 1 − P ( x ^ | x )

onde x a instncia a ser prevista e x ^ a previso mais provvel. Ou seja, ele pega uma amostra com a menor probabilidade de pertencer a determinada classe. (a mais incerta!). Para saber mais, acesse a documentao do modAL.

N_QUERIES = 94
unqueried_score = learner.score(X_raw, y_raw)
performance_history = [unqueried_score]

X_pool_uncert = X_pool.copy()
y_pool_uncert = y_pool.copy()
from sklearn.base import clone

lr2 = clone(learner, safe=True)
# Allow our model to query our unlabeled dataset for the most
# informative points according to our query strategy (uncertainty sampling).
for index in range(N_QUERIES):
    query_index, query_instance = lr2.query(X_pool_uncert)

    # Teach our ActiveLearner model the record it has requested.
    X_act, y_act = X_pool_uncert[query_index].reshape(1, -1), y_pool_uncert[query_index].reshape(1, )
    lr2.teach(X=X_act, y=y_act)

    # Remove the queried instance from the unlabeled pool.
    X_pool_uncert, y_pool_uncert = np.delete(X_pool_uncert, query_index, axis=0), np.delete(y_pool_uncert, query_index)

    # Calculate and report our model's accuracy.
    model_accuracy = lr2.score(X_raw, y_raw)
    print('Accuracy after query {n}: {acc:0.4f}'.format(n=index + 1, acc=model_accuracy))

    # Save our model's performance for plotting.
    performance_history.append(model_accuracy)

    # Save our model's performance for plotting.
